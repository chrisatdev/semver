# Semantic Version

This script creates semantic versions and also the badges

## Usage

```
$ ./semver.sh [Options] [Args]
```

    Options:
      -b       Base 
      -M       Major
      -m       Minor
      -p       Patch
      -c       Custom 
      -h       Help

    Args (Optional):
      dev
      stable
      x.x.x   Only for custom
